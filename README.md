# 🔐 Password Manager Python

Simple gestionnaire de mots de passe développé en Python.

##  Fonctionnalités

- Ajouter un compte
- Afficher tous les comptes
- Rechercher un compte
- Supprimer un compte
- Masquer les mots de passe sans authentification
- Sauvegarder les données dans un fichier Excel

##  Technologies utilisées

- Python
- openpyxl
- getpass

##  Structure

Le programme permet de gérer plusieurs comptes avec :

- Plateforme
- Nom d'utilisateur
- Mot de passe

Les mots de passe sont protégés par un mot de passe général.
